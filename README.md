# virtual_mouse
This is a university project for our "advanced computer vision" class.

# Requirements
### Python packages
mediapipe 0.10.14 \
opencv-python 4.10.0.84 \
numpy 2.1.2 \
pyautogui 0.9.54 \
OneEuroFilter 0.2.1


# Usage
1. Make sure a webcam is connected to your device
2. Open main3.0.py and adjust the necessary parameters
3. Run main3.0.py
